# miloop.id - E-Commerce Rajutan Premium
**Proyek UAS KAIT II - Administrasi Bisnis | Semester Genap 2025/2026**

## 1. Business Overview
**miloop.id** adalah platform e-commerce fungsional yang berfokus pada produk rajutan (*knitwear*) *handmade* premium. Kami memadukan seni merajut tradisional dengan gaya fashion *vintage* dan *aesthetic* masa kini.

*   **Value Proposition:** Produk *handmade* eksklusif yang dibuat dengan ketelitian tinggi, memberikan nilai emosional dan unik bagi setiap pelanggan.
*   **Target Market:** Wanita usia 16–28 tahun (Pelajar, Mahasiswa, *First-jobber*) yang menyukai gaya *cottagecore* dan menghargai produk lokal.

## 2. Strategi Bisnis & Manajemen
*   **Model Bisnis:** B2C (*Business to Consumer*) melalui kanal *Direct-to-Consumer* berbasis website.
*   **Revenue Stream:** Penjualan langsung produk fisik (Amigurumi, Bucket Hat, Tote Bag, Keychain).
*   **Strategi Harga:** *Value-based pricing*, di mana harga ditentukan berdasarkan tingkat kerumitan perajutan dan nilai seni eksklusif produk.
*   **Manajemen Produk:** Katalog terorganisir dalam 4 kategori utama dengan deskripsi produk yang menarik untuk memperkuat *branding*.
*   **Payment Gateway:** Simulasi sistem pembayaran terintegrasi (QRIS, BCA, Mandiri) untuk mempermudah alur transaksi pelanggan.

## 3. Integrasi Data Analytics
Website ini menggunakan pendekatan berbasis data untuk pengambilan keputusan bisnis. Metrik yang dipantau meliputi:
1.  **Bounce Rate:** Mengukur efektivitas *Hero Banner* dalam menahan pengunjung agar tidak langsung keluar.
2.  **Conversion Rate:** Persentase pengunjung yang berhasil melakukan *checkout* sukses.
3.  **Cart Abandonment Rate:** Memantau pengguna yang menambahkan barang ke keranjang namun tidak menekan tombol pembayaran (digunakan untuk evaluasi promosi/biaya kirim).

## 4. Technical Explanation
*   **Tech Stack:** Pure HTML5, CSS3, dan JavaScript (Vanilla) untuk performa optimal dan SEO-friendly[cite: 6].
*   **Responsive Design:** Menggunakan **CSS Flexbox dan CSS Grid** yang terjamin tampil sempurna di ukuran layar *Mobile, Tablet,* hingga *Desktop*[cite: 6].
*   **Storage:** Fitur keranjang, stok real-time, dan riwayat pesanan memanfaatkan API **LocalStorage** sehingga data tersimpan secara lokal tanpa perlu *backend* server[cite: 6].
*   **Fitur Utama:** Sistem autentikasi, manajemen stok real-time, dasbor admin dengan grafik penjualan (Chart.js), dan validasi form[cite: 6].
*   **Deployment:** Proyek dideploy menggunakan **GitHub Pages**.
